By Ward Hasan · June 2025

So, I decided to open up an old project — an app I started working on about a year ago

Due to my long-standing habit of not staying organized with long-term stuff (yes, I’m owning it), things got a little messy

(And yes, I tend to go into details before getting to the point — so buckle up and dive in with me)

Here’s what I mean by “not being organized”: When I’m actively working on something, everything feels clear. I know exactly how to navigate it. But once I step away for too long? It’s like I’ve never seen it before. That’s why I’ve recently started adopting proper documentation habits — finally

Anyway, circling back to the main topic: I found the project! But... when I tried to run it, MySQL threw an error on my system 8)

And instead of fixing it like a responsible dev, I hyper-fixated on creating this blog instead. :DDDD!!!

Looking at the bright side — we improved our footer! It's now dynamically loaded as a reusable component

We added this placeholder in our HTML page:

<div id="footer-placeholder"></div>

brings up the idea that you can load react builds here if needed <3

And this script to handle rendering the footer (we called it load-partials.js):


          function loadHTML(selector, url) {
            fetch(url)
              .then(response => {
                if (!response.ok) throw new Error(`Failed to load ${url}`);
                return response.text();
              })
              .then(html => {
                document.querySelector(selector).innerHTML = html;
                const shouldShowCards = document.body.classList.contains('showFooterCards');
                const footerCards = document.querySelector('#footer-cards-wrapper');
                if (footerCards) {
                  footerCards.style.display = shouldShowCards ? 'block' : 'none';
                }
              })
              .catch(err => console.error(err));
          }

          document.addEventListener("DOMContentLoaded", () => {
            const placeholder = document.querySelector("#footer-placeholder");
            if (!placeholder) return;

            const bodyId = document.body.id;
            let url;

            if (bodyId === "home-page") {
              url = "partials/footer.html";
            } else {
              url = "../partials/footer.html";
            }

            loadHTML("#footer-placeholder", url);
          });
